Why do we need git ?

  • To keep track of all the changes.
  • It provides version control system i.e. we can revert back to older version of code at anytime.

Git Commands :

  • First initialize your folder with - git init
  • Add your files to the staging area - git add .
    • . is to represent all files in the current directory
    • * is to add all files including sub-folders
    • We can add individual file using git add file_name
  • How to set credentials?
    • set email - git config --global user.email "your git email"
    • set name - git config --global user.name "your beautiful name"
  • See commit history using - git log
  • git status - is used to get the status of your directory i.e. which files are staged, untracked, modified etc
